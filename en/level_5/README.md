## 🛠️ New Tools

### Phoenix

```bash
mix phx.new my_app --live
cd my_app
mix ecto.create
mix phx.server
```

Visit: http://localhost:4000

### Nerves (with Raspberry Pi)

```bash
export MIX_TARGET=rpi4
mix nerves.new my_firmware
mix firmware
mix burn
```
