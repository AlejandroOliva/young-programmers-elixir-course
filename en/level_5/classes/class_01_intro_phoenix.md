# 🌐 Class 1: Introduction to Phoenix

## 🎯 Objectives

* 🌐 Learn Phoenix Framework
* 🚀 Create first web app
* 📋 Basic MVC structure

## 🚀 Create Phoenix Project

```bash
mix phx.new my_app --live
cd my_app
mix ecto.create
mix phx.server
```

Visit: http://localhost:4000

## 🎯 Phoenix Structure

```
my_app/
├── lib/
│   ├── my_app/           # Business logic
│   └── my_app_web/       # Web layer
├── assets/               # Frontend
├── priv/                 # Static assets
└── test/
```
