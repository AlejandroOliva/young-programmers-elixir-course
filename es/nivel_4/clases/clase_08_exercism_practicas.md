## 🌐 Exercism Setup

```bash
# Instalar CLI
# Ver: https://exercism.org/cli-walkthrough

# Descargar ejercicio
exercism download --exercise=two-fer --track=elixir

# Trabajar en el ejercicio
cd ~/exercism/elixir/two-fer

# Ejecutar tests
mix test

# Subir solución
exercism submit two_fer.ex
```
