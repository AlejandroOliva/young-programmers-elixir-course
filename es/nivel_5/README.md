## 🛠️ Herramientas Nuevas

### Phoenix

```bash
mix phx.new mi_app --live
cd mi_app
mix ecto.create
mix phx.server
```

Visita: http://localhost:4000

### Nerves (con Raspberry Pi)

```bash
export MIX_TARGET=rpi4
mix nerves.new mi_firmware
mix firmware
mix burn
```
